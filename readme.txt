=== Description ===

The plugin displays your posts in Slider.

Features include:
* It can be controlled by admin
* you can use shortcode in any page in your site
* Built-in WordPress locale codes
* display the posts, photo and text in Slider

= Installing manually: =

1. Unzip all files to the `wp-content/plugins/a-slider` directory
2. Log into WordPress admin and activate the 'Nice Slider' plugin through the 'Plugins' menu
3. Go to *Nice Slider > Home* in the left-hand menu to take shortcode to use him in any page in your site
